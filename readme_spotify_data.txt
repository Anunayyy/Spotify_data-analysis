here we downloaded the tracks datasetfrom kaggle.
this dataset consists ofsongs from 1922 to 2021.
the songs have a special key for identification.
the attributes of the song are popularity,artist,dacebility,loudness and a lot more.
here we performed data analysis on this dataset to get a lot of important results, on how the songs changed overtime.
how to duration, dancebility etc changed in the last 100 years.
we have used numpy,matplotlib,pandas and seaborn for our analysis.
the code for the analysis is present with the dataset in this respository.
the code is free to use to extract even more important results.
